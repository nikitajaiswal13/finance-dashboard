# Finance Dashboard (FinTrack)

## Overview
A simple finance dashboard to track income, expenses, and overall balance with visual insights.

## Features
- Dashboard with total balance, income, expenses
- Balance trend chart
- Spending category chart
- Transactions list with filtering
- Role-based UI (Admin / Viewer)
- Insights section
- Responsive design

## Tech Stack
- Angular (TypeScript)
- Angular Material
- Chart.js

## Live Demo
[Click here](https://fintrackboard.netlify.app/dashboard)

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `ng serve`
